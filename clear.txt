针对极为变态的小小小小小小鸡，运行这些命令，可以提高一丢丢内存

1. 软件包深度卸载：
apt-get purge -y dbus* vim* xauth bash-completion dialog xdg-user-dirs shared-mime-info publicsuffix isc-dhcp-client isc-dhcp-common net-tools ncurses-term krb5-locales libc-l10n locales libx11-* libxcb1 && apt-get autoremove -y && apt-get clean

2. 物理文件强制清理：
rm -rf /usr/share/doc/* /usr/share/man/* /usr/share/info/* /usr/share/locale/*

3. 然后看看效果
df -h /

只能用这个方法安装singbox
# 1. 确保在 /tmp 目录下（利用内存）
cd /tmp

# 2. 流式下载并提取，直接存入 /usr/bin/sing-box
curl -L https://github.com/SagerNet/sing-box/releases/download/v1.10.7/sing-box-1.10.7-linux-amd64.tar.gz | tar -zxv --strip-components=1 -C /usr/bin/ sing-box-1.10.7-linux-amd64/sing-box

# 3. 立即赋予权限
chmod +x /usr/bin/sing-box
